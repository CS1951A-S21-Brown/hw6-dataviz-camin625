WRITTEN QUESTION RESPONSES


1)
My first graph displays each genre along the y-axis with bars extending from left to right. The length of the bar indicates the number of titles that fall under that specific genre. To view the exact count of titles under a specific genre, one can hover over the text that reads "hover here." Furthermore, the graph is sorted by counts so that we can compare the genres with the highest and lowest title counts.

My second graph is formatted in a similar way to the first. The release year is located on the y-axis with bars extending from left to right and longer bars indicating longer average runtime for that release year. The graph starts by displaying the 15 release years with the longest average runtime, but the graph has an interactive element where users can pick if they want to view the top 5, 10, 15, 20, or 25 release years, and the graph is adjusted accordingly. The graph is sorted by average runtime so that we can view the years with the longest average runtime.

My third graph is a line plot that contains actor-director duos along the y-axis. The further to the right, that a specific duo's point extends, the more movies they have worked on together. Furthermore, this graph also contains a hovering feature because it can be difficult to read the duo's names otherwise. Using a line plot allows users to view which duos have the most movies together quite easily especially with the hover feature. For viewing reasons, I decided to display the top 50 actor-director duos.
